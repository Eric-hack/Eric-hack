** [Live API](https://alx-project-nexus-aaao.onrender.com)** | ** [API Docs](https://alx-project-nexus.onrender.com/api/docs/)** | ** [Repository](https://github.com/Eric-hack/alx-project-nexus)**

A production-ready RESTful API built with Django REST Framework, featuring:

###  Key Features
-  **JWT Authentication** - Secure user registration and login
-  **Product Management** - Full CRUD with advanced filtering, search & pagination
-  **Order Processing** - Complete checkout workflow with inventory management
-  **Shopping Cart** - Real-time cart management
-  **Product Reviews** - User ratings and feedback system
-  **Database Optimization** - Indexed queries with `select_related` & `prefetch_related`
-  **Interactive API Docs** - Swagger/OpenAPI documentation
-  **Docker Deployment** - Containerized for easy deployment

###  Tech Stack
- **Backend:** Django 5.0, Django REST Framework
- **Database:** PostgreSQL with optimized queries & indexes
- **Authentication:** JWT (Simple JWT)
- **API Docs:** drf-yasg (Swagger/OpenAPI)
- **Deployment:** Docker, Render (Cloud Platform)
- **Version Control:** Git with semantic commits
